Välkommen till Förintelsens Tidslinje

Denna interaktiva tidslinje visar viktiga händelser under perioden 1933-1946, från Nazitysklands framväxt till andra världskrigets slut.

Hur du använder tidslinjen:

  • Navigera i tiden: Använd pilknapparna längst ner eller tryck på prickarna på tidslinjen för att röra dig mellan olika tidsperioder.
  • Utforska händelser: Händelser visas som cirklar på kartan. Hovra över dem för att se en förhandsvisning, klicka för att läsa mer.
  • Läs kontext: Den vita rutan längst ner till höger ger historisk kontext för varje år.
  • Tangentbordsnavigering: Använd vänster/höger piltangenter för att navigera.
